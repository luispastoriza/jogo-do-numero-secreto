## Jogo-do-Numero-Secreto
Jogo simples desenvolvido em JavaScript que sorteia um número aleatório e o usuário deve tentar acertar qual número foi sorteado.  

## Funcionalidades
- Sortear número aleatório
- Verificar respostas do usuário e dar feedbacks
- Possui a biblioteca ResponsiveVoice.js para adicionar feedback em áudio, tornando a experiência mais interativa

## Tecnologias
- HTML
- CSS
- JavaScript

## Objetivo
Praticar lógica de programação, condicionais, laços de repetição e interação com o usuário
